# Personas

## Usuário Principal

### Perfil
Pessoa que busca uma solução simples, intuitiva e rápida para utilização da plataforma.

### Objetivos
- Resolver tarefas rapidamente
- Encontrar informações facilmente
- Utilizar o sistema sem dificuldade

### Dores
- Interfaces confusas
- Lentidão
- Excesso de informações

---

## Administrador

### Perfil
Responsável pelo gerenciamento e monitoramento da plataforma.

### Objetivos
- Organizar informações
- Garantir funcionamento adequado
- Gerenciar usuários e conteúdos

### Necessidades
- Painel administrativo simples
- Facilidade de manutenção
- Boa visualização de dados
